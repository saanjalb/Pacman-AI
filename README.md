🎮 Pac-Man AI Chase
A Pac-Man inspired game built with Python and Pygame where ghosts use the A* pathfinding algorithm to intelligently chase the player through a randomly generated maze.
🧠 AI Features

Ghosts use A* (A-Star) pathfinding to find the shortest route to Pac-Man in real time
Maze is randomly generated every game for a unique experience each time

🕹️ How to Play

Use Arrow Keys to move Pac-Man
Collect all 10 pellets to win
Avoid the red ghost — it will chase you using AI!
You have 3 lives — losing all means Game Over

🛠️ Tech Stack

Language: Python
Library: Pygame
Algorithm: A* Pathfinding

▶️ How to Run

Install dependencies:

   pip install pygame

Run the game:

   python pacman.py
